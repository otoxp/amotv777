# IPTV Copilot

Painel IPTV moderno, seguro e responsivo. Suporte a listas M3U, busca de canais, player HLS.js e navegação por teclado.

## Funcionalidades
- Carregamento via URL ou arquivo M3U
- Filtro por grupo e busca por nome
- Player com suporte a `.m3u8`
- Interface responsiva e acessível
- Destaque de canal ativo
- Salvamento do último canal assistido
